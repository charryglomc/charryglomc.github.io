缓冲区

  • 定义

    • 内存的一部分

  • 作用

    • 速度、空间不匹配的中间产物

  • 类型

    • 全缓冲

      • 例如：对磁盘文件读写

    • 行缓冲

      • 例如：键盘的输入操作

    • 不带缓冲

      • 例如：cerr

  • 输出缓冲区

    • 缓冲区内容被刷新几种情况

      • 程序正常结束

      • 缓冲区满

      • 使用操作符显示刷新缓冲区

        • flush
        • endl

      • 使用unitbuf设置流状态以及nounitbuf回到正常状态

      • 输入流与输出流相关联

        • tie

文件输入流ifstream

  • 当文件不存在时，打开文件失败

    • ifstream ifs

      • 构造函数多种，需要选取

  • 从文件读数据并输出到屏幕

    • ifs >> word

      • 默认以空格分隔

  • 从文件一次读取一行使用getline函数

    • while(getline(ifs, line))

    • 如果想把读出的数据保存起来

      • string line[128]

      • vector

        • 讲解其原理

文件输出流ofstream

  • 默认情况，当文件不存在的时候创建文件，当文件已经存在，则清空文件再写

    • ofstream ofs

      • 构造函数多种，需要选取

文件输入输出流fstream

  • 使用fstream这个类进行输入输出

    • 举例：从终端输入数据然后存到文件，之后从文件读数据然后输出到屏幕(都使用fstream)

      • 查找文件指针的位置

        • tellp/tellg

      • 设置文件指针的位置

        • seekp/seekg

          • std::ios::beg
          • std::ios::cur
          • std::ios::end

文件的模式

  • 输入模式

    • in输入

    • ate

      • 文件指针最初在文件末尾

        • 等同于在in模式下，使用std::ios::end

  • 输出模式

    • out输出

    • app追加

      • 写入始终在文件末尾
