CSS文字文本样式

字数统计: 908 字 | 阅读时长 ≈ 3 分钟

摘要

本文主要介绍:css字体font的一些样式属性、外观属性、chrome开发者工具和sublime快捷操作emmet语法

CSS字体样式属性调试工具

1. font字体

1.1 font-size：字号

  • 由于不同浏览器默认显示的字体大小可能不同，所以我们在设置字体大小时应该给定明确值
  • 谷歌浏览器的默认字体大小是16px

1.2 font-family:字体

  • 字体名中含有空格，$，#等，要用英文状态下的双引号
  • 中文字体要用英文状态下的双引号
  • 英文不区分大小写

字体名称英文名称Unicode 编码

注意：Unicode字体也要用引号包起来

1.3 font-weight:字体粗细

  • 在html中我们使用b和strong标签来设置字体粗细
  • 而在CSS中：
属性值描述
normal默认值（400）
bold加粗（700）
number100-900

1.4 font-style:字体风格

  • 在html中我们使用em标签哎设置字体风格
  • 而在CSS中
属性值描述
normal正常
italic斜体

1.5 font：综合设置字体样式

注意：

  • 上面的书写顺序要严格遵循
  • font-size和font-family一定要写，其他可不写（取默认值），否则font属性将不起作用

2. CSS外观属性

2.1 color：文本颜色

属性值含义
预定义的颜色值red,green,blue等
十六进制#000000黑色，#ffffff白色，#ff0000红色,#ccc灰色等
RGB三原色rgb(255,0,0)

一般十六进制用的多一些，且可采用简写：如#ffffff 可写为#fff

2.2 text-align：文本水平对齐方式

属性值文本水平对齐方式
left左对齐（默认）
center居中对齐
right右对齐
  • 注意：是设置盒子里内容的水平对齐方式，而不是盒子水平对齐方式

2.3 line-height：行间距

  • 一般只设置为比字号大7-8px左右就可以了

2.4 text-indent：文本缩进

  • 单位为em
  • 1em就是一个字的宽度，如果是汉字，1em就是一个汉字的宽度，一般我们缩进2em

2.5 text-decoration：文本装饰

属性值含义
none文本默认，也可以用来取消下划线
underline给文本加下划线（a标签链接默认）
overline给文本加上划线
line-through给文本加删除线

3. chrome开发者工具

  • ctrl+滚轮 ：可放大缩小开发者页面
  • 左侧是html元素结构，右侧是css样式
  • 右侧css样式可以改动数值和颜色并动态查看效果
  • ctrl + 0 复原开发者页面大小
